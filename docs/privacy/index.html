Privacy Policy

Last updated: June 23, 2026

Shorts Speed Controller is a browser extension that adds playback speed controls to YouTube Shorts.

Data Collection

The extension does not collect, transmit, sell, rent, or share personal data.

Local Storage

The extension stores only one setting: the selected playback speed. This value is saved locally through Chrome extension storage so the same speed can be restored after page reloads and used across Shorts tabs.

Network Requests

The extension does not send data to external servers and does not use analytics, advertising SDKs, or remote code.

Website Access

The extension is loaded on YouTube pages so it can detect YouTube's single-page navigation into Shorts. It only adds the speed control interface and applies the selected playback speed when the current page is a YouTube Shorts page.

Changes

This policy may be updated if the extension behavior changes. Updates will be published on this page.

Contact

For privacy questions, use the support contact listed on the Chrome Web Store listing for this extension.
